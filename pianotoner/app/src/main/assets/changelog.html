Version History

The initial development of this small app began in the year 2018, as an iOS version with advertising.

Since then, lots of things have changed, I made an Android version, almost 99% similar (not really in the graphical aspect, but in functionality),

in Juli 2020 I released the last closed source version, and finally in February 2021 I decided to opt in favour of Open Source code and publish the source code on GitHub.
